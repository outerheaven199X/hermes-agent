# Krea 1

Krea's proprietary model. Real-time iteration, avoids typical "AI look". Accurate skin textures, dynamic camera angles, expressive styles.

## When to Use

- Iterative art direction (see changes in real time)
- Avoiding the "plastic AI" look
- Artistic and expressive rendering
- Need to sketch/layout before generating
- Style transfer or enhancement
- Quick exploration of multiple directions

## Core Advantage

Real-time canvas: see images evolve as you type/sketch. Iteration is live art direction, not prompt roulette.

## Available Models in Krea

- **Krea 1**: Stylish photographic, avoids cheesy AI look (default)
- **Flux variants**: Flux Krea, Flux Kontext
- **Nano Banana**: Via Krea interface
- **ChatGPT Image**: Via Krea interface
- **Ideogram 3.0**: Text-heavy designs

## Prompt Components

1. **Subject**: Main focus (person, object, character)
2. **Action**: What's happening
3. **Style**: Artistic approach, medium
4. **Environment**: Setting details

## Krea 1 Characteristics

- Perfect imperfection: grain, bloom, blur
- Dreamy, vivid, weird
- Artistic and expressive rendering
- Natural skin textures (not plastic AI look)

## Image Prompts

Upload reference images to influence generation. Model combines:
- Characters from reference
- Style elements
- Color/composition cues

Best in: Krea 1, Flux, Nano Banana, ChatGPT Image, Ideogram 3.0

## Layout Control (Nano Banana, ChatGPT Image only)

Sketch rough layout to influence composition. Helpful for unconventional framing.

## Resolution

Some models support 1.5x, 2x, 4x size (up to 30 megapixels).

## Distilled Mode

On/off toggle that reduces model's default aesthetic preferences. Use when you want raw interpretation without "prettifying."

## Real-Time Canvas

- Sketch shapes to guide AI
- Type and watch output evolve
- Annotate to nudge direction

## Tips

- More detailed prompts → better AI interpretation
- Experiment with word combinations
- Use custom AI models for brand consistency
- Leverage community/tutorials for technique refinement

## Editing Features

- In/out-painting
- Add/remove objects
- Style transfer
- Enhancer (upscale to 22K)

## Video Generation

Access multiple video models through Krea: Veo 3, Kling, Hailuo, Wan, Runway Gen-4

## Common Pitfalls

1. **Accepting first output** — The canvas is meant for iteration. Keep refining
2. **Too much distillation** — Distilled mode can produce raw/unfinished looks
3. **Ignoring sketch input** — Even rough sketches dramatically improve composition control
4. **Wrong model for task** — Use Ideogram for text-heavy, Nano Banana for reasoning, Krea 1 for artistic
